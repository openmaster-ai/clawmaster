# 龙虾管理大师——OpenClaw 生态的六边形战士

## Overview
龙虾管理大师（ClawMaster）是 OpenClaw 生态的一站式智能管理平台，由开源社区协作共建，提供六大核心能力和四大生态支柱，解决用户在安装、监控、降本、应用、开发和安全方面的六大痛点。

## Learning Objectives
The viewer will understand:
1. 龙虾管理大师的六大核心能力分别是什么，各解决什么痛点
2. 四大生态伙伴（OceanBase、百度、LangChain、算力魔方）如何支撑平台
3. 六大核心与生态之间的映射关系，以及产品的整体定位

## Section 2: 六大核心能力 (六个辐射节点)

**Key Concept**: 六大核心能力分别解决一个用户痛点

**Content**:

### 节点1 - 能接管
- 痛点：装不上
- 解法：帮你装、帮你管、一切从这里开始
- 关键功能：一键安装 + 自动接管，OpenClaw / LangChain 全家桶

### 节点2 - 能观测
- 痛点：看不清
- 解法：花了多少、跑得如何、一目了然
- 关键功能：本地代理 + 可视化 Dashboard + LangSmith 对接

### 节点3 - 能省钱
- 痛点：花太多
- 解法：PowerMem + seekdb，Token 降 96%
- 关键数据：Token 降 96%，准确率反升 48%

### 节点4 - 能应用
- 痛点：用不起来
- 解法：拍照答题/错题本/发票整理，开箱即用
- 关键功能：通过飞书/微信/钉钉直接触达

### 节点5 - 能构建
- 痛点：造不出来
- 解法：跟管家聊天，智能体就建好了
- 关键功能：基于 LangChain DeepAgents，零代码零门槛

### 节点6 - 能守护
- 痛点：不放心
- 解法：密钥加密、花费熔断、权限管控
- 关键功能：API Key 加密 + 花费熔断 + 操作审计 + 100% 本地

**Visual Element**:
- Type: 六个发光节点围绕中心六边形均匀分布
- Subject: 每个节点是一个圆角卡片，带霓虹边框，内含图标+核心名称+痛点+解法
- Treatment: 不同颜色霓虹光（青蓝/品红/紫/橙/绿/金），辐射线从中心连接到各节点

**Text Labels**:
- 节点1: "能接管" / "装不上→帮你装"
- 节点2: "能观测" / "看不清→一目了然"
- 节点3: "能省钱" / "花太多→Token 降 96%"
- 节点4: "能应用" / "用不起来→开箱即用"
- 节点5: "能构建" / "造不出来→对话即创建"
- 节点6: "能守护" / "不放心→安全可控"

## Section 3: 四大生态支柱 (底部横向排列)

**Key Concept**: 四大开源力量各司其职，协作支撑平台

**Content**:

### 支柱1 - OceanBase · 数据层
- seekdb AI 原生数据库
- PowerMem 记忆引擎
- 全量数据存储
- 映射核心：能省钱

### 支柱2 - 百度 · 模型层
- PaddleOCR 强力 OCR
- 基础模型推理能力
- 映射核心：能应用

### 支柱3 - LangChain 社区 · 编排层
- DeepAgents 深度智能体
- LangGraph 工作流引擎
- LangSmith 可观测
- 映射核心：能构建 + 能观测

### 支柱4 - 算力魔方 · 硬件层
- 开源硬件标准
- 内置龙虾管家
- 边缘算力
- 映射核心：全平台支撑

**Visual Element**:
- Type: 底部四个等宽卡片横向排列
- Subject: 每个卡片带品牌 logo 占位 + 角色名 + 核心能力列表
- Treatment: 深色卡片带微弱霓虹边框，与上方核心节点有虚线映射连接

**Text Labels**:
- 支柱1: "OceanBase · 数据层" / "seekdb + PowerMem"

## Design Instructions

### Style Preferences
- 科技感、专业
- 暗色背景 + 霓虹发光效果 (cyberpunk-neon)
- 青蓝/品红/紫色为主调
- 适合开源社区传播

### Layout Preferences
- Hub-spoke: 中心龙虾 → 六大核心辐射
- 底部横向排列四大生态支柱
- 上方六大核心与下方生态之间有映射虚线

### Other Requirements
- 中文内容
- 16:9 横版
- 适合 GitHub README、社交媒体、PPT 引用
- Slogan 醒目置顶或居中
