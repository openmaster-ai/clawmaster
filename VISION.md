# 🦞 龙虾管理大师 (ClawMaster) —— 产品定位

> **龙虾管理大师——OpenClaw 生态的六边形战士：能接管、能观测、能省钱、能应用、能构建、能守护。**

龙虾管理大师（ClawMaster）是 OpenClaw 生态的一站式智能管理平台，由开源社区协作共建，为每一位用户提供六大核心能力：接管、观测、省钱、应用、构建、守护。

---

## 为什么需要龙虾管理大师？

| 痛点 | 现状 | 龙虾管理大师的解法 |
|------|------|-------------------|
| **装不上** | OpenClaw / LangChain 安装配置门槛高，很多人卡在第一步 | 一键安装 + 自动接管，总入口解决一切 |
| **看不清** | Token 消耗、API 费用、上下文健康度全是黑盒 | 本地代理 + 可视化 Dashboard，一目了然 |
| **花太多** | Token 烧钱如流水，完整历史反复灌上下文 | PowerMem + seekdb，Token 降 96%，准确率反升 48% |
| **用不起来** | 装完不知道能干什么，缺少即用场景 | 内置拍照答题、错题本、发票整理等开箱即用技能 |
| **造不出来** | 智能体开发要写代码，普通人玩不转 | 跟管家对话即可创建/迭代智能体，零代码零门槛 |
| **不放心** | API Key 明文、费用失控、无权限隔离 | 密钥加密、花费熔断、操作审计、权限管控 |

## 六大核心能力

### 核心一：能接管 —— "帮你装、帮你管、一切从这里开始"

龙虾管理大师是 OpenClaw 生态的总入口。无论你是零基础新手还是已有安装的用户，管家都能帮你搞定。

- **环境检测**：自动检测 Node.js / npm / OpenClaw / Python / LangChain 全链路
- **一键安装**：引导式安装 OpenClaw + LangChain 全家桶
- **接管现有安装**：发现已装好的环境，读取配置，纳入管理
- **配置可视化编辑**：所有配置可查看、修改、保存
- **网关启停控制**：GUI 一键启动/停止/重启 Gateway
- **通道管理**：添加/编辑/删除飞书、微信、钉钉、Telegram 等通道
- **模型提供商管理**：管理 API Key、默认模型、备选链

### 核心二：能观测 —— "花了多少、跑得如何、一目了然"

别的方案要装 ClawProbe + cc-switch + LangSmith 一堆工具才能凑齐的能力，龙虾管家一站式搞定。

- **本地代理 (Local Proxy)**：本地 HTTP 代理拦截所有 LLM API 请求，零侵入采集数据
- **费用追踪 Dashboard**：日/周/月费用，按模型/提供商/Agent 分组，趋势图表
- **Token 用量分析**：每次调用 input/output/cache token 计数，累计统计
- **上下文健康度**：窗口占用率、压缩事件时间线、静默截断检测
- **会话管理器**：本地会话历史浏览、搜索、完整对话查看、一键恢复
- **智能告警**：费用飙升、上下文快满、压缩过频、工具描述截断等预警
- **LangSmith 对接**：代理层导出 trace，专业级调用链追踪与评估

### 核心三：能省钱 —— "PowerMem + seekdb，Token 降 96%"

核心原因是每轮对话都把完整历史当上下文灌进去，大量 Token 浪费在重复信息上。通过 PowerMem 智能记忆管理 + seekdb 一体化存储，只检索相关记忆注入上下文。

- **PowerMem 集成**：智能提取对话关键事实，Ebbinghaus 遗忘曲线管理记忆生命周期
- **seekdb 一体化存储**：统一向量搜索 + 全文检索 + 关系查询，替代默认 SQLite
- **记忆可视化管理**：GUI 浏览/搜索/编辑/删除 Agent 记忆，查看记忆层级
- **遗忘曲线可视化**：展示记忆衰减状态，哪些快要遗忘、哪些被强化
- **多 Agent 记忆隔离**：每个 Agent 独立记忆空间，支持跨 Agent 搜索
- **省钱报告**：对比开启 PowerMem 前后的 Token 消耗，量化节省金额
- **提供商性价比分析**：基于历史数据推荐更省钱的模型/提供商方案

### 核心四：能应用 —— "拍照答题、错题本、发票整理，开箱即用"

用户装完最常问的一句话："然后呢？能干什么？"。龙虾管家内置实用技能，通过飞书/微信/钉钉等国内主流办公前端直接触达用户，装完就能用，不需要二次开发。

- **拍照答题**：拍照 → OCR → 解题 → 返回答案
- **错题本**：自动收集错题、分类归档、定期推送复习，结合 PowerMem 长期记忆
- **发票整理**：拍照/转发 → 自动识别抬头金额类型 → 归档 → 导出报表
- **技能市场**：浏览/安装/更新第三方 Skills
- **前端通道支持**：飞书、微信、钉钉等国内主流平台已全部支持接入 OpenClaw

### 核心五：能构建 —— "跟管家聊天，智能体就建好了"

传统智能体开发要写代码、配环境、调参数，普通用户根本玩不转。基于 LangChain DeepAgents，用户只需用自然语言对管家描述需求，管家自动完成智能体的创建、调试和迭代。

- **对话式创建智能体**：用自然语言描述需求，管家调用 DeepAgents 自动生成 Agent
- **对话式迭代修改**：对已有智能体提出变更需求，管家自动执行
- **DeepAgents 运行时**：Python sidecar 运行 LangChain DeepAgents
- **LangGraph 工作流**：复杂多步任务的状态机驱动流程
- **LangSmith 评估**：自动评估新建/修改后的智能体质量
- **未来生态扩展**：对接 Dify、Flow 等外部 Agent 平台

### 核心六：能守护 —— "密钥加密、花费熔断、权限管控"

API Key 明文存储、多人无权限隔离、费用异常无熔断、敏感数据无加密——用户越深入使用，安全焦虑越大。龙虾管家守住安全底线。

- **API Key 加密保管**：所有密钥加密存储，UI 脱敏显示
- **花费熔断机制**：设定日/周/月预算上限，触及阈值自动暂停 API 调用
- **操作审计日志**：谁在什么时候做了什么操作，全部可追溯
- **多用户权限管控**：管理员/操作员/只读角色，不同权限不同视图
- **数据本地化保障**：所有数据 100% 本地，零云端依赖，零遥测上报
- **敏感信息扫描**：自动检测配置/日志中意外暴露的 Key、Token、密码

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 桌面框架 | Tauri 2.x | 比 Electron 轻量 10 倍，使用系统 WebView |
| 前端框架 | React 18 + TypeScript | 生态成熟，组件库丰富 |
| UI 组件 | Shadcn/ui + Tailwind CSS | 现代、可定制、体积小 |
| 状态管理 | Zustand | 轻量、简单 |
| 双模式 | Desktop (Tauri IPC) + Web (Vite + Express) | 一套代码，两种运行方式 |
| 数据存储 | seekdb (生产) / SQLite (开发) | AI 原生数据库 |
| 记忆引擎 | PowerMem | 智能长期记忆，Token 降 96% |
| 智能体框架 | LangChain DeepAgents + LangGraph | 无代码深度智能体构建 |
| OCR 能力 | PaddleOCR | 百度开源强力 OCR |
| 可观测 | LangSmith + 本地代理 | 专业级调用链追踪 |
